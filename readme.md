# SOAP Multilanguage Case Study

Caso de estudio para el consumo de un servicio web SOAP utilizando múltiples lenguajes de programación del lado del servidor.

## Lenguajes

- Node.js
- Java
- Go
- .NET 10
- Ruby
- Perl
- C++
- Rust

## Estructura

Cada lenguaje contiene tres versiones:

1. Cliente SOAP.
2. Cliente SOAP con traducción al español.
3. Aplicación web que convierte números a letras en español.
